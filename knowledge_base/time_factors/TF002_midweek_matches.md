# Midweek Match Dynamics

## Tuesday/Wednesday (CL/EL)
- European matches = maximum effort
- League matches can be rotated

## League Matches Midweek
- Often faster pace (less tactical)
- More goals on average
- Tired teams make errors

## Recovery Windows
| Gap | Impact |
|-----|--------|
| 3 days | -10% performance |
| 4 days | -5% performance |
| 5+ days | Baseline |
