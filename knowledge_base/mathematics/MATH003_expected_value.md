# Expected Value (EV) Mastery

## Formula
EV = (Probability × Profit) - (1 - Probability) × Stake
EV = (P × (Odds - 1)) - (1 - P)

## Minimum EV Thresholds
- Single bets: >3% EV
- Accumulators: >5% per leg
- High volume: >2% acceptable

## EV Edge Compounding
- 1000 bets at +3% EV = expected +30 units
- Variance requires 500+ bets for convergence
