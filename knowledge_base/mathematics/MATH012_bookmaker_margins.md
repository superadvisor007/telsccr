# Bookmaker Margin Analysis

## How to Calculate
Sum of implied probabilities - 100% = Margin

## Typical Margins
| Bookmaker Type | Margin |
|----------------|--------|
| Sharp (Pinnacle) | 2-3% |
| Mainstream | 4-6% |
| Retail shops | 8-12% |

## Value Hunting
- Compare across bookmakers
- Sharp books = best prices
