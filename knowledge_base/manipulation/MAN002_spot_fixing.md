# Spot Fixing Patterns

## Common Spot Fix Targets
- First throw-in (team to take)
- First corner timing
- First card timing
- Number of corners in specific period
- Time of first goal

## Detection
- Unusual betting volume on minor markets
- Specific player behavior patterns
- Timing irregularities

## Safe Markets
- Match result (hardest to fix)
- Over/Under goals (difficult to control precisely)
- BTTS (requires both teams' participation)
