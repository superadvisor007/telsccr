# New Manager Bounce

## Statistical Pattern
| Period | Win Rate Change |
|--------|-----------------|
| First match | +15% |
| Matches 2-5 | +10% |
| Matches 6-10 | +5% |
| After 10 | Returns to baseline |

## Caretaker vs Permanent
- Caretaker: Short boost, no long-term plan
- Permanent: Initial dip possible (new system)

## Manager Style Change
- Defensive → Offensive: Goals increase weeks 3-6
- Offensive → Defensive: Clean sheets improve immediately
