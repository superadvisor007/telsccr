# Pattern: Over 1.5 Goals in High-Scoring Leagues

## Win Rate: 78-82%
## Optimal Odds: 1.35-1.45

## Target Leagues
| League | Over 1.5 Rate |
|--------|---------------|
| Bundesliga | 78.7% |
| Eredivisie | 80.9% |
| Premier League | 78.4% |
| Serie A | 78.5% |

## Entry Criteria
- Both teams scored in last 3 matches
- Combined xG > 2.5
- Not a defensive derby
- Odds > 1.35

## Exit Rules
- If odds drop below 1.30, skip
- If both teams changed managers, skip
