# Home Crowd Psychology

## Statistical Impact
- Home teams win 46% vs 27% away (historical average)
- Referee decisions favor home team by 7-12%
- Empty stadiums reduce home advantage to 2-4%

## Crowd Size Effects
| Attendance % | Home Advantage Boost |
|--------------|---------------------|
| >95% | +15% |
| 70-95% | +10% |
| 50-70% | +5% |
| <50% | +2% |

## Hostile Atmospheres
- Derby matches: crowd effect doubles
- Relegation battles: crowd pressure increases errors
